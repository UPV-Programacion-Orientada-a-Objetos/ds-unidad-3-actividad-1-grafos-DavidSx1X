\doxysubsection{Detailed Description}
Implementación optimizada de grafos usando CSR (Compressed Sparse Row). 


\begin{DoxyItemize}
\item Esta clase gestiona la memoria manualmente mediante vectores compactos para permitir la carga de millones de nodos en RAM limitada. 
\end{DoxyItemize}
