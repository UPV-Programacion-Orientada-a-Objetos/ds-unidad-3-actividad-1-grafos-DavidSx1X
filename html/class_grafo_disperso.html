Detailed Description

Implementación optimizada de grafos usando CSR (Compressed Sparse Row).

  • Esta clase gestiona la memoria manualmente mediante vectores compactos para permitir la carga de millones de nodos en RAM limitada.

void GrafoDisperso::cargarDatos ( const std::string & filename)
overridevirtual

Carga el grafo. Decide inteligentemente entre Texto o Caché Binaria.

  • Algoritmo:
  1. Verifica si existe la versión .bin del archivo.
  2. Si existe, carga directamente los vectores a RAM (milisegundos).
  3. Si no, lee el .txt línea por línea, construye el CSR y genera la caché.
